SolarSim

An interactive simulation of the solar system, built with Angular and Three.js. Includes all 200+ moons, the 5,000 brightest celestial objects, and 9 constellations.

Angular Three.js WebGL TypeScript

Project Overview

SolarSim is a comprehensive solar system visualization tool that brings the cosmos to your browser. Key features include:

  • Accurate 3D models of all planets and major moons
  • Real-time orbital mechanics simulation
  • Interactive camera controls with zoom and pan
  • Detailed information panels for celestial bodies
  • Constellation visualization with star mapping
  • Time controls for speeding up/slowing down orbital motion

Built with performance in mind, the application uses WebGL through Three.js for smooth rendering of complex 3D scenes, while Angular provides a robust framework for the user interface and state management.

SolarSim (Unity version)

A physics-driven solar system simulation built with the Unity game engine, featuring gravity simulation for dynamic and chaotic interactions. Developed as a school project.

Unity C# Physics Gravity simulation

Project Overview

This Unity-based version of SolarSim focuses on accurate physics simulation of celestial bodies. Key features include:

  • Real-time gravity calculations between all bodies
  • Customizable mass and velocity settings
  • Dynamic orbit visualization
  • Interactive camera system
  • Ability to create and destroy celestial bodies
  • Time control system for simulation speed

The project demonstrates the capabilities of Unity's physics engine while providing an educational tool for understanding orbital mechanics and gravitational interactions.
